# Plantilla — Pulse financiero mensual

> ⚠️ PLACEHOLDER — Este archivo será reemplazado por el contenido real.

Plantilla que uso cada mes en Enzo Motorsport y Arkhē Coworking. Formato semáforo para detectar desviaciones rápido.

## Estructura

### 1. Headline (1 línea)
```
[Mes] [Año] — [Negocio] · 🟢 / 🟡 / 🔴
```

### 2. Top-line metrics
| Métrica | Mes | Target | Status |
|---------|-----|--------|--------|
| Revenue | $X | $Y | 🟢/🟡/🔴 |
| Margen bruto | X% | Y% | |
| Cash en cuenta | $X | $Y | |

### 3. Desviaciones
Solo lo que está fuera de target. Una línea por desviación con causa probable.

### 4. Acciones para [Próximo Mes]
Máximo 3 acciones concretas con responsable y deadline.

### 5. Decisiones tomadas este mes
Link a tu decision log.

## Reglas

- Si toma más de 30min hacerlo cada mes, está mal estructurado
- Si todo es verde, agrega un risk que estés watching
- Comparte con tu board/asesores el día 5 del mes siguiente
- Archiva — el patrón mes a mes es el insight
